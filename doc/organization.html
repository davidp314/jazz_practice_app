Jazz Guitar Practice Tracker

A task-centered approach to structured jazz guitar practice with flexible goals and collaborative sharing.

Core Structure

Practice Organization Hierarchy:

Goals (flexible duration: one session to months)
  ├── Practice Session 1
  │   ├── Work on: Autumn Leaves
  │   ├── Work on: Scale practice
  │   └── Work on: Repertoire review
  ├── Practice Session 2
  │   └── Work on: Autumn Leaves (continued)
  └── Practice Session 3
      └── Work on: Record performance video

Tasks: The Fundamental Unit

Tasks are persistent practice objectives that evolve over time. Each task represents something you're working on that might span multiple practice sessions.

Example Task: "Autumn Leaves"

Type: Jazz Standard  |  Key: Bb major  |  Status: Active

Goals:

  • Learn basic chord progression
  • Practice shell voicings
  • Work on arpeggios in bridge section
  • Practice improvisation over changes
  • Record performance video

Practice Details: iReal Pro backing track, Medium Swing, Focus on voice leading

Task Types

Jazz Standards

Traditional jazz compositions with suggested goal templates (chord progressions, scales, improvisation, etc.)

Repertoire

Any completed piece that needs maintenance - jazz standards, classical pieces, technical exercises

Other Work

Technical exercises, transcriptions, theory study, equipment practice

Flexible Goals

Instead of rigid step-by-step algorithms, tasks have customizable goals. When creating jazz standard tasks, users can choose from common goal templates or create their own:

Beginner Template

Basic chord progression → Melody → Combine → Performance

Intermediate Template

Shell voicings → Scales → Voice leading → Improvisation → Performance

Custom Goals

Create specific objectives tailored to your learning needs and teacher's approach

Practice Sessions

A Practice Session represents a single sitting at your instrument - what you actually do when you sit down to practice. Sessions consist of work on multiple tasks and can last anywhere from 20 minutes to several hours.

Example Practice Session: "Tuesday Evening Practice"

Total Time: 60 minutes  |  Date: January 15, 2025

Work on tasks in this session:

  • Autumn Leaves - work on shell voicings (25 min)
  • Scale warmups - major scales (15 min)
  • Blue Bossa - repertoire review (10 min)
  • Metronome practice - timing exercises (10 min)

Session Focus: Voice leading and rhythm

Session Characteristics

Task Persistence

The same task (like "Autumn Leaves") appears across multiple sessions as you dedicate time to working toward its goals

Time Management

Allocate time to work on each task, with flexible overtime tracking when you're in the zone

Session Notes

Record discoveries, challenges, and breakthroughs while working on each task during practice

Progress Tracking

Mark task goals as completed during the session based on your work, updating your overall progress

Goals

Goals are overarching objectives that provide direction and motivation for your practice. They span multiple practice sessions and can range from short-term projects to long-term musical development.

Example Goal: "Master Autumn Leaves for Jazz Ensemble Audition"

Target Date: March 15, 2025  |  Priority: High  |  Status: In Progress

Success Criteria:

  • Play melody and chords confidently
  • Demonstrate solid voice leading
  • Improvise coherently over the changes
  • Perform at 140 BPM with backing track
  • Record audition video

Associated Tasks: Autumn Leaves, ii-V-I exercises, Bebop scale practice

Progress: 2 of 5 criteria completed, 8 practice sessions logged

Goal Information

Objective & Timeline

Clear description of what you want to achieve and when, with flexible duration from days to months

Success Criteria

Specific, measurable outcomes that define goal completion - performance standards, technique milestones, repertoire additions

Associated Tasks

Link existing tasks to the goal, showing how daily practice connects to bigger objectives

Priority & Context

Importance level, motivation, and context (gig preparation, lesson assignment, personal growth)

Goal Tracking

Progress Metrics

Completion percentage, time invested, practice sessions dedicated to goal-related tasks

Milestone Recognition

Celebrate achievements along the way - first clean playthrough, successful improvisation, performance readiness

Reflection Notes

Record insights, challenges overcome, and lessons learned throughout the goal journey

Accountability

Due date reminders, progress reports, and motivation tracking to maintain momentum

Practice Approaches

Quick Practice Session
Start Session → Select tasks to work on → Optional focus → Practice

Minimal planning for spontaneous practice. Select tasks to work on and optionally set a session focus to encourage intentionality without being pushy.

Goal-Based Practice
Create Goal → Plan Sessions → Execute Sessions → Complete Goal

Structured approach for working toward specific objectives over days, weeks, or months.

Package-Based Practice
Import Package → Follow Session Sequence → Complete Package

Use pre-designed practice curricula from teachers or other musicians.

Practice Packages

Groups of related practice sessions that can be shared between users. Anyone can create packages - teachers assigning work, advanced players sharing transcription studies, or personal practice curricula.

Teacher Assignments

Multi-session curricula with dependencies, due dates, and progress tracking

Community Content

Shared practice sequences for specific techniques, styles, or repertoire

Personal Programs

Self-created practice series for focused skill development

Key Design Principles

Flexible Structure: Accommodates different learning styles from spontaneous practice to rigid curriculum following.

Task Persistence: Practice objectives evolve over multiple sessions, reflecting how real learning works.

Goal Customization: No rigid algorithms - adapt objectives to your needs and teacher's approach.

Collaborative Learning: Share practice sequences and learn from other musicians' approaches.

Progressive Enhancement: Start simple, add structure and detail as you see value.
