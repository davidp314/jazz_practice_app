General Introduction

Overview

The Jazz Guitar Practice Tracker is a sophisticated web-based practice management system specifically designed for jazz guitarists who want to approach their learning systematically and track their progress over time. Unlike generic practice apps, this tool understands the unique challenges of jazz education and provides features that align with proven jazz pedagogy.

Why Jazz Guitar Needs a Specialized Tool

Learning jazz guitar involves simultaneously developing multiple skills:

  • Harmonic Knowledge: Understanding complex chord progressions and substitutions
  • Technical Skills: Mastering fingerings, scales, arpeggios, and voice leading
  • Improvisational Abilities: Creating melodic lines and developing jazz vocabulary
  • Repertoire Management: Learning and maintaining dozens of standards

This application addresses these challenges with a structured, step-by-step approach while maintaining the flexibility that creative practice requires.

User Tutorial

Getting Started

Initial Setup

  1. Launch the Application: Open the app in any modern web browser. No installation or account creation required.
  2. Choose Your Theme: Click the sun/moon icon in the top-right to toggle between light and dark modes.
  3. Add Your First Standards:
    • Click "Add Standard" in the Jazz Standards section
    • Enter the name of a jazz standard you're working on (e.g., "Autumn Leaves")
    • The system will create a new standard with all 8 learning steps unchecked
  4. Add Other Practice Work:
    • Click "Add Other Work" to include technical exercises, etudes, or transcription work
    • Enter a name and optional description

Understanding the 8-Step Learning Algorithm

Each jazz standard follows a proven learning progression:

  1. Play through with staple chords - Get familiar with the basic progression
  2. Learn shell chords - Master essential voice leading
  3. Learn scales for each chord - Understand harmonic choices
  4. Learn arpeggios for each chord - Build melodic vocabulary
  5. Target the 3rds - Develop smooth voice leading
  6. Comping - Practice rhythm guitar and accompaniment
  7. Practice improvisation - Apply your knowledge creatively
  8. Post performance video - Document your achievement

Managing Your Practice Materials

Working with Standards

  • Check off completed steps by clicking the checkboxes next to each step
  • Toggle Active/Inactive status to focus on current priorities
  • Mark as Completed when you've mastered all 8 steps
  • Add notes to record insights, challenges, or performance dates
  • Use filters to view Active, All, or Completed standards

Managing Other Work

  • Create entries for technical exercises, etudes, transcription work, or equipment practice
  • Use the same Active/Inactive and Completed toggles for organization
  • Add descriptions to remind yourself of specific goals or techniques

Editing Existing Items

  • Click "Edit Standard" or "Edit Other Work" buttons
  • Select the item you want to modify from the list
  • Update names, notes, completion status, or step progress
  • Save changes to update your records

Planning Practice Sessions

Session Setup Process

  1. Start a New Session: Click "Start Session" from the overview page
  2. Add Standards to Practice:
    • Select from your active standards in the dropdown
    • Each standard gets a default 25-minute allocation
    • Expand tasks to select a specific focus step (optional)
    • Add practice notes for your session goals
  3. Include Other Work:
    • Choose from your active other work items
    • Default allocation is 20 minutes per item
    • Add session-specific notes about your focus
  4. Add One-Off Tasks:
    • Create custom tasks for today's session only
    • Examples: "Warm-up exercises", "Jam session prep", "Equipment setup"
    • Set your own time allocation
  5. Repertoire Review (Recommended):
    • The app suggests completed standards that need review
    • Follows the 20% rule (20% of total session time for maintenance)
    • Suggests the least-recently-practiced completed standard
    • You can override suggestions and select different repertoire pieces
  6. Organize Your Session:
    • Drag and drop tasks to reorder them
    • Adjust time allocations as needed
    • Expand individual tasks to add detailed practice notes
    • Review total session time before starting

During Practice Sessions

Basic Timer Usage

  1. Start Your Session: Click "Start Practice Session" after setup
  2. Select Current Task: Click on any task to make it active
  3. Use Timer Controls:
    • Click Play/Resume to start timing
    • Click Pause to stop timing
    • Use spacebar for quick play/pause toggle

Advanced Practice Features

Smart Task Switching
  • Click any task to switch focus instantly
  • Timer pauses automatically when switching
  • Previous task time is preserved in memory
  • Resume any task exactly where you left off
Flexible Overtime
  • Continue working beyond allocated time when inspired
  • Overtime is tracked and displayed clearly
  • Visual indicators show tasks that are over their target time
  • Total session time includes all overtime
Keyboard Navigation
  • Arrow Keys: Switch between tasks (Up/Down or Left/Right)
  • Spacebar: Play/pause current task timer
  • Task Indicators: Visual feedback for current, completed, and overtime tasks

The main screen provides at-a-glance information:

Detailed Reports

Click "Reports" to access comprehensive analytics including weekly statistics, monthly totals, and detailed session history with notes and reflections.

Data Management

Backup and Restore

  • Export Data: Click "Export" to download a complete backup file (JSON format)
  • Import Data: Click "Import" to restore from a backup file
  • Data Migration: Safely transfer between devices or browsers

Data Models

Standard Object Schema

{
  id: string,                    // Unique identifier
  name: string,                  // Standard name (e.g., "Autumn Leaves")
  type: 'standard',              // Object type identifier
  steps: boolean[8],             // 8-step completion tracking
  completed: boolean,            // Overall completion status
  active: boolean,               // Currently working on this
  notes: string,                 // User notes and observations
  lastWorkedOn: Date            // Last practice date
}

Other Work Object Schema

{
  id: string,                    // Unique identifier
  name: string,                  // Exercise/work name
  type: 'other_work',           // Object type identifier
  description: string,           // Detailed description
  completed: boolean,            // Completion status
  active: boolean,              // Currently working on this
  lastWorkedOn: Date           // Last practice date
}

Practice Session Schema

{
  id: string,                    // Unique session identifier
  date: Date,                    // Session start date/time
  tasks: Task[],                 // Array of session tasks
  totalTime: number,             // Planned total time (minutes)
  completed: boolean,            // Session completion status
  endTime: Date,                 // Actual end time
  taskTimeSpent: {               // Actual time spent per task
    [taskId]: seconds            // Task ID mapped to seconds
  },
  finalNotes: object,            // Final session notes per task
  stepCompletions: object        // Completed steps during session
}

The timer system implements several sophisticated features:

  • Overtime Handling: Continues tracking beyond allocation gracefully

Data Persistence System

The application uses localStorage for client-side persistence with automatic saving and data migration handling:

useEffect(() => {
  localStorage.setItem('jazzStandards', JSON.stringify(standards));
}, [standards]);

const migrateStandardsData = (standardsData) => {
  return standardsData.map(standard => {
    if (standard.steps.length === 7) {
      // Insert new "comping" step at index 5
      const newSteps = [...standard.steps];
      newSteps.splice(5, 0, false);
      return { ...standard, steps: newSteps };
    }
    return standard;
  });
};

Repertoire Rotation Algorithm

const getRepertoireRotation = () => {
  const activeCompletedStandards = standards.filter(s => s.completed && s.active);
  
  return activeCompletedStandards.sort((a, b) => {
    const dateA = a.lastWorkedOn ? new Date(a.lastWorkedOn) : new Date(0);
    const dateB = b.lastWorkedOn ? new Date(b.lastWorkedOn) : new Date(0);
    return dateA - dateB; // Oldest first
  })[0];
};
